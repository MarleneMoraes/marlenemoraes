## Indexation Professionnelle

<img src="./assets/MarleneMoraes_por_NanciYin.png" alt="Marlene par Nanci Yin" width="150px" align="right"/>

### **[C] Cloud et Infrastructure**
- **Note d'application :** Orchestration de services distribués et automatisation des pipelines de déploiement dans des environnements de haute disponibilité.
- **Trajectoire :**
    - *Cloud Public :* Maîtrise d'AWS (Lambda, S3, API Gateway, IAM) et GCP (Google Cloud Platform).
    - *Cycle DevOps :* Création de pipelines CI/CD via Jenkins et gestion de version avec Git et Mercurial.

### **[G] Génie Logiciel**
- **Note d'application :** Application de méthodes systématiques au développement de logiciels, en se concentrant sur les modèles de conception (design patterns), l'intégrité des données et l'évolutivité des systèmes.
-  **Trajectoire :**
    - *Écosystème Java :* Développement d'API RESTful et de microservices avec Java 17+ et Spring Boot.
    - *C# & .NET Core :* Expérience axée sur les règles métier complexes, les couches de sécurité et la prévention de la fraude.
    - *Programmation Système :* Maintenance de bibliothèques internes et de services de recherche en C++11.

### **[S] Sciences de l'Information**
- **Note d'application :** Champ interdisciplinaire axé sur la collecte, la classification, la manipulation, le stockage et la récupération de l'information.
- **Trajectoire :**
    - *Mémoire Sociale :* Recherche académique sur la construction d'espaces de mémoire dans le Hip Hop brésilien (UFMG).
    - *Gestion des Connaissances :* Application des principes des sciences de l'information dans l'architecture logicielle pour transformer les données en connaissances récupérables.
